Class: Collection

Collection(name)

A Simple collection that stores names and a Metric instances with a few convenience methods for creating / registering and then gathering all data the registered metrics.

Constructor

new Collection(name)

Creates a named collection of metrics
Parameters:
Name Type Description
name string The name to use for this collection.
Source:
  • lib/Collection.js, line 20
Example
var { Collection } = require('measured');
const collection = new Collection('node-process-metrics');
const gauge = collection.gauge('node.process.heap_used', () => {
   return process.memoryUsage().heapUsed;
});

  • lib/Collection.js, line 48
Example

const collection = new Collection('node-process-metrics');

   return process.memoryUsage().heapUsed;
});
collection.register('node.process.heap_used', gauge);
